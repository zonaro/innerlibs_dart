Creates a ResponsiveRow with a specific number of columns in each ScreenTier and automatically wraps children into ResponsiveColumn widgets. If a child is already a ResponsiveColumn, it remains unchanged. If a child is not a Widget, this function converts it to a Text widget and then wraps it in a ResponsiveColumn.
