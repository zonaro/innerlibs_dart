Creates a FloatingArea widget.

The children parameter is a list of draggable children. The background parameter is an optional widget that appears behind the draggable children. The onDragEnd parameter is a callback function that is called when a draggable child is released. The onDragUpdate parameter is a callback function that is called when a draggable child is being dragged. The foreground parameter is an optional widget that appears in front of the draggable children.
